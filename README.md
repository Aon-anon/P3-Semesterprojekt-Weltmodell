# Projektdemonstration im Rahmen P3 WS 25/26 IMD hda

Ich habe einen dist Ordner erstellt und diesen im Repo hochgeladen. Um auf diesen zuzugreifen, lade den Code als zip Datei runter, extrahiere diese und öffne ein Terminal im Hauptordner des Projekts.
